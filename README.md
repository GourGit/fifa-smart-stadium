# ⚽ Smart Stadium AI Assistant – FIFA World Cup 2026

A **Generative AI-powered Smart Stadium Assistant** built for **Hack2Skill Prompt Wars Challenge 4**. The platform enhances the FIFA World Cup 2026 experience for both **fans** and **stadium staff** by providing AI-powered assistance, live match updates, public information, and an intelligent command center.

---

## 🚀 Features

### 👤 Fan Assistant

* 🤖 AI chatbot for stadium and match-related queries
* ⚽ Live match scores and schedules
* 📍 Stadium navigation and venue information
* 🚍 Live transportation and public information
* 🌦️ Weather updates
* 🌐 Multilingual AI assistance (future scope)

### 🏟️ Staff Command Center

* 📊 Real-time operational dashboard
* 🧠 AI-powered decision support
* 👥 Crowd monitoring insights
* 🚨 Live stadium updates
* ⚡ Faster coordination for venue staff

# ⚙️ Installation

## 1. Clone the Repository

```bash
git clone https://github.com/your-username/your-repository.git
```

```bash
cd your-repository
```

---

## 2. Install Frontend Dependencies

```bash
cd client
npm install
```

---

## 3. Install Backend Dependencies

```bash
cd ../server
npm install
```

---

## 4. Configure Environment Variables

Create a `.env` file inside the **server** folder.

```env
PORT=5000

GENAI_API_KEY=YOUR_API_KEY

SPORTS_API_KEY=YOUR_API_KEY

PUBLIC_DATA_API_KEY=YOUR_API_KEY
```

---

## ▶️ Running the Application

### Start the Backend

```bash
cd server
npm start
```

or

```bash
npm run dev
```

---

### Start the Frontend

Open another terminal.

```bash
cd client
npm start
```

or (if using Vite)

```bash
npm run dev
```
